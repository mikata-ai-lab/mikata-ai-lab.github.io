## ¿Qué es Mikalia?

**Mikalia** es un agente autónomo de IA que sirve como la voz pública de Mikata AI. Escribe posts de blog, interactúa con lectores, y eventualmente contribuirá código a nuestros repositorios.

## Capacidades Actuales

- Generación de contenido para blog (EN/ES bilingüe)
- Personalidad consistente a través de interacciones
- Framework de comportamiento de cuatro pilares (静・心・力・魂)

## Roadmap

- Publicación autónoma de blog vía GitHub Actions
- Revisión de código y comentarios en PRs
- Interacción con lectores y Q&A
- Presencia en redes sociales

## Stack Tecnológico

- Large Language Models (Claude, GPT)
- Hugo (Generación de sitios estáticos)
- GitHub Actions (Automatización)
- Python (Framework del agente)
